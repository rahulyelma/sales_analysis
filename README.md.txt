# 📊 Sales & Profit Analysis Dashboard (Excel)

## 🔍 Overview

This project analyzes sales performance using Excel Pivot Tables, Slicers, and Charts.

## 📁 Dataset

* Date
* Sales
* Cost
* Quantity
* Selling Price

## 📈 Key Metrics

* Total Sales
* Total Profit
* Profit %
* Monthly Trends

## 🧠 Insights

* Profit margin ranges between 50%–60%
* Some days have high sales but lower efficiency
* Monthly trends show stable performance

## 🛠 Tools Used

* Excel (Pivot Tables, Slicers, Pivot Charts)
